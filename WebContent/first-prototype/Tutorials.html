Zeppelin notebook

Deeplearning4j offers interactive tutorials for the features of the library. If you haven't set up zeppelin visit this site's "First steps" and follow how to set up zeppelin.

Downloading notebooks:

Click on "Download this notebook" on the tutorial you would want to interact with and then copy the URL link you are forwarded to: https://raw.githubusercontent.com/deeplearning4j/dl4j-examples...
Then go to your browzer and visit localhost:8080 (where zeppelin is). Click on "import note" -> "Add from url". Paste you'r coppied url in the URL section and lastly "Import note"

Use a notebook:

Go to zeppelin and click on the drop menu "Notebook".When you ran the command "docker ...skymindops/zeppelin-dl4j:latest..." zeppelin comes with the first 8 tutorials (Excluding "Quickstart for Deeplearning4j"), so you don't have to download these 8 tutorials manually.
