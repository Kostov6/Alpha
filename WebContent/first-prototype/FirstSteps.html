First steps

I bet you have heard of the so-called AI or artificial intelligence. Are you curious about learning more about it? Given you have some experience in coding or are willing to start learning more about software engineering, AI might be a little bit of an advanced topic but don't worry! Take it as a challenge and use this website as a helping hand in learning about coding and AI.

Before you start

It is a good idea to have some knowledge on the basics on how to code. If you haven't started to program, we encourage you to begin a ready-to-use course on programming or if you are more of a self-learned we can recommend you some free online tutorials and books for those who prefer reading.
Our choice in language is JAVA because this language is constructed around a concept called Object Oriented Programming(OOP). This concept is quite intuitive and when you get used to it, understanding of programming in general will become easier.

Github:

Github is a source control software or in simpler words - a database for source files. For beginners, the only feature that is important for you is the download source function. From the so called repository(the project you want do download) click on "Clone or download" then "Download zip". This will download the source files. Extract them and then open them from files->Open project from file system from your IDE (in this example it is Eclipse)
